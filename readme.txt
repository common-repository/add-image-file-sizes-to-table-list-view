Get the file sizes of media files, add the sizes to the media table/list view as a column, and make it sortable.

== Description ==

The "Add Image File Sizes to Table List View" plugin enhances your WordPress media library by adding a new column that displays the file sizes of your images. This feature allows you to quickly assess the space your media is taking up and helps in managing your media files more effectively. The plugin also makes this new column sortable, enabling you to organize your media library by file size easily.

== Installation ==

1. Upload the `add-image-file-sizes-to-table` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Once activated, navigate to your media library in list view mode to see the new file size column.

== Frequently Asked Questions ==

= Can I hide the file size column if needed? =

Yes, you can toggle the visibility of the file size column from the screen options tab in the top right corner of the media library page.
